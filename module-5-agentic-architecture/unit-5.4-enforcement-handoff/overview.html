Unit 5.4 — Enforcement & Handoff Patterns

Task Statement 1.4 — Choose programmatic enforcement vs prompt-based guidance; design structured handoff summaries

⏱ 2 min read → then open the Lab

Decision Tree: Programmatic vs Prompt-Based

This decision appears in almost every exam scenario involving safety, compliance, or financial rules.

QuestionIf YESIf NO
Must the rule be followed 100% of the time? Programmatic enforcement (hooks, prerequisite gates, code-level checks) Prompt-based guidance is acceptable
Are there financial/legal/safety consequences for violation? MUST be programmatic — prompts have a non-zero failure rate Prompt guidance for best practices, style preferences
Is the rule a hard constraint (refund limits, identity verification)? Prerequisite gate: block the action until the condition is verified in code Prompt instruction with monitoring

Structured Handoff Summaries

When control passes between agents (or from agent to human), the handoff summary must be structured, not narrative.

Include in HandoffExclude from Handoff
What was accomplished (structured list of actions taken) Internal reasoning or chain of thought
Current state (verified facts, pending items) Speculative analysis
What remains (explicit next steps for the recipient) Full conversation history
Any constraints or blockers discovered Tool call details or raw API responses
💡 Key Insight: Prompt instructions have a non-zero failure rate. For rules with financial, legal, or safety consequences, this failure rate is unacceptable. The exam tests this distinction heavily: if the scenario involves money, compliance, or safety, the answer is always programmatic enforcement (hooks, gates, code checks) — never "add it to the prompt."
⚠ Exam Trap: Answer options like "add a system prompt instruction telling the agent to always verify identity before processing refunds" sound reasonable but are wrong for critical rules. The correct answer is a prerequisite gate: code-level check that blocks the refund tool unless identity verification has been completed. Prompts guide; code enforces.
